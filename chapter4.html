Chapter 4: Programming Techniques


There are 3 basic constructs that have remained the same with computing and algorithms. These are sequence, selection and iteration:

Sequence

There isn't a lot to sequence. This is the execution of statements or functions one after another. The majority of code will usually involve just sequences of instructions.


Selection

This is where a decision is made in a program and control is passed to another point in the program. The decision is based on a Boolean (a data type that is either true or false, 1 or 0, etc) expression. In high level languages this is seen with IF statements and ELSE statements.In low level languages such as LMC (Little Man Computer), this decision making is achieved with branching such as BRA or BRZ.
